Features

Everything contextkeeper does

12 capabilities. One tool. Zero context loss across any AI model.

CLI Tool

Full project lifecycle from the terminal. Five core commands.
Details

contextkeeper init - Initialize project
contextkeeper sync - Capture session state
contextkeeper bootstrap - Load state into new session
contextkeeper status - View project state
contextkeeper doctor - Health check + diagnostics

Python SDK

Typed models, Pydantic validation, programmatic session management.
Details

from contextkeeper import Session
Full type hints for IDE autocomplete. Pydantic-validated models for state vectors, decisions, and invariants. Async support for high-throughput pipelines.

MCP Server

Native Claude integration. Registered on the official MCP registry.
Details

Claude can read and write session state directly through the Model Context Protocol. No copy-paste, no CLI - Claude talks to contextkeeper natively. Listed on the official Anthropic MCP registry.

REST API

FastAPI-powered. 17 endpoints. OpenAPI spec. API key auth.
Details

17 RESTful endpoints covering sessions, decisions, invariants, bundles, and status. OpenAPI/Swagger documentation auto-generated. API key authentication with role-based access control.

State Machine Governance

10 states. Enforced transitions. Protocol breach detection.
Details

DEL-v2 defines 10 lifecycle states from UNINITIATED to SEALED. Every transition is validated. Illegal jumps trigger PROTOCOL_BREACH, halting all operations. Fail-closed by design. See the full diagram.

Dual-SHA Tracking

Two SHA256 hashes per sync. Authority + repo integrity.
Details

Every sync captures both an authority hash (authoritative file content) and a repo hash (git HEAD). On bootstrap, both are verified. Divergence without a recorded transition flags an integrity violation.

Deterministic Handoff Bundles

Self-contained ZIP with SHA256 manifest and full repo snapshot.
Details

14-step pipeline generates a portable ZIP containing: state vector, decision log, invariant registry, file manifest, execution evidence, and SHA256 manifest. Hand it to any AI, any developer, any auditor.

Invariant Registry

Named formal properties. Machine-checkable assertions.
Details

Define constraints like "p99 latency under 200ms" or "no direct DOM manipulation in React." Each invariant is named, scoped, and traced to the session that established it. Gate checks enforce them.

Multi-Agent Coordination

Operator / Executor / Auditor role separation.
Details

No single AI can self-approve. Operators define tasks, Executors implement, Auditors verify. Role separation prevents unauthorized state transitions. See team workflows.

Multiple Backends

File, SQLite, and PostgreSQL storage backends.
Details

Start with flat-file JSON for simplicity. Scale to SQLite for single-machine performance. Move to PostgreSQL for team/enterprise deployments. Same API, different storage.

Protocol Breach Detection

Fail-closed. Halt on violation. Require human recovery.
Details

Illegal state transitions, SHA mismatches, invariant violations, and unauthorized role actions all trigger immediate halt. The system will not auto-recover. Human intervention required.

CI/CD Integration

GitHub Actions. Gate enforcement in your pipeline.
Details

Run contextkeeper doctor in CI to validate state integrity. Enforce gate checks before merge. Block deployments if invariants are violated. Works with GitHub Actions, GitLab CI, or any runner.
