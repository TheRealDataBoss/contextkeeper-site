FAQ

Common questions

Is contextkeeper really free?

Yes. The core tool - CLI, Python SDK, MCP server, file and SQLite backends, handoff bundles, unlimited sessions - is free forever. Open source under MIT license. No credit card, no trial period, no feature gates on the core product.

Can I self-host everything?

Yes. contextkeeper is fully self-hostable. The Free tier runs entirely on your machine with no external dependencies. Team and Enterprise plans add cloud features, but you can always run the core tool locally or on your own infrastructure.

What's the difference between Free and Pro?

Pro adds cloud sync (access your state from any machine), REST API access for custom integrations, priority bootstrap speeds, an analytics dashboard showing session history and decision trends, and email support.

Do I need a paid plan for team use?

Not necessarily. Teams can share STATE_VECTOR.json through their git repo on the Free tier. The Team plan adds managed shared state vectors, role-based access control, team decision logs, PostgreSQL backend, and Slack notifications.

Is contextkeeper open source?

Yes. The core is MIT-licensed and available on GitHub. Paid plans add managed infrastructure and enterprise features on top of the open source core.
